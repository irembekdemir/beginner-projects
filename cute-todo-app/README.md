# Cute To-Do List Application ✨

A charming, colorful desktop To-Do List application built with Python. 
It features a modern user interface and allows users to export their daily tasks into a beautifully formatted, color-coded Excel spreadsheet directly to their desktop.

## 🌸 Features

- **Charming UI:** Designed with a colorful, eye-catching palette using `CustomTkinter`.
- **Dynamic Task Management:** Easily add and remove tasks with simple checkboxes.
- **Smart Excel Export:** Exports your tasks into a structured, custom-styled Excel sheet with proper fonts, colors, and auto-adjusted column widths.
- **Status Tracking:** Automatically marks tasks as "Completed ✨" or "Pending ⏳" in the exported Excel file based on your checkboxes.
- **Lightweight & RAM Friendly:** Built to run efficiently within a virtual environment.

## 🛠️ Built With

- **Python 3.14**
- **CustomTkinter** - For the modern, smooth UI and widgets.
- **OpenPyXL** - For creating and styling the Excel files.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Make sure you have Python installed on your system. 

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/irembekdemir/todo_list_app.git](https://github.com/irembekdemir/todo_list_app.git)
   cd todo_list_app.py
   ```

2. **Create a virtual environment ( Recommended, RAM Friendly :D ):**
    ```bash
    python -m venv venv
    ```

3. **Activate the virtual environment:**

Windows:

    ```bash
    venv\Scripts\activate
    ```

Mac/Linux:

    ```bash
    source venv/bin/activate
    ```
    
4. **Install the required dependencies:**

    ```bash
    pip install customtkinter openpyxl
    ```

5. Run the application:

    ```bash
    python todo_app.py
    ```
